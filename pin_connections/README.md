# Raspberry Pi 5 GPIO Connections

**SparkFun BME280**
- GPIO 1 (3V3)  --> 3V3
- GPIO 30 (GND) --> GND
- GPIO 2 (SDA)  --> SDA
- GPIO 3 (SCL)  --> SCL

**Adafruit BNO085**
- GPIO 1 (3V3)  --> VIN
- GPIO 39 (GND) --> GND
- GPIO 2 (SDA)  --> SDA
- GPIO 3 (SCL)  --> SCL
